Nori Shaders BE

Nori shader is one of the Minecraft be shaders that have PBR features and blocky clouds because it uses(have) PBR it is certain this shader also has a feature like the reflection on the block, bump map and also I added a step parallax mapping with self shadow




Changelog

here are some changes from the previous beta version

  • some updates on the sky
  • new tonemap
  • new ray marching blocky clouds
  • new pbr mapping method
  • support PBR with LabPBR format
  • improve step parallax mapping
  • improve ggx specular
  • more..

Shader Settings

to make some changes in the shader go to root directory of this shader > shaders > glsl > shader_settings.txt
You can also change the time cycle or position of sun


PBR

by default the PBR feature in this shader is off / nonactive and to activate it requires an additional resource pack, here the resource pack is specially made for nori shader to activate it and as become base texture of PBR mapping in nori shader
this resource pack is installed on top of the shader and then above it is installed a resource pack that has the PBR feature


don't forget to set the texture resolution on the nori pbr base resource pack according to the pbr texture resolution used on it



for texture pack creator

This shader use minecraft Java edition PBR with oldPBR or labPBR format and don't create or add terrain_texture.json in the texture pack
here are the requirements:

  • each texture pack must have a PBR configuration file for nori shaders and be stored in the root directory texture pack > shaders > glsl > nori_pbr_set.txt

  • texture grass_side_n and grass_side_s must be duplicated 16x and starting from grass_side0_n/_s, no grass_side_n/_s
    and also if there is grass_side_snowed.tga and grass_side_snowed.png in this one directory, delete one of them

  • each texture flipbook must have the same resolution (frame) as the normal texture map and specularity


Note !

  • This shader is only recommended for high-end devices
  • This shader is not support minecraft bedrock windows 10 32bit (not renderdragon version)
  • This shader is not playable and the time cycle is not matched with minecraft default time, so this shader is not survival friendly.
  • Don't reupload the shader file, if you want to share it, just use this github page link
  • Don't make this github page link or download link as a shortlink and activate some payment method on the shortlink
