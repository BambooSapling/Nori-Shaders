Nori Shaders BE

             


Nori shader is one of the Minecraft be shaders that have PBR features and blocky clouds because it uses(have) PBR it is certain this shader also has a feature like the reflection on the block, bump map and also I added a step parallax mapping along with the self shadow
This shader is not related to minecraft default time because all the lighting elements in this shader use atmospheric scattering, so an accurate value of the sun's position is needed. Here I tricked it by creating a fake position (value) that can be set by the user.

Shader Settings

to make some changes in the shader go to root directory of this shader > shaders > glsl > common.glsl
You can also change the time cycle or position of sun


PBR

by default the PBR feature in this shader is off / nonactive and to activate it requires an additional resource pack, here the resource pack is specially made for nori shader to activate it and as become base texture of PBR mapping in nori shader
this resource pack is installed on top of the shader and then above it is installed a resource pack that has the PBR feature



for texture pack creator
